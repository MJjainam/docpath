# DocPath

Generate and serve documentation for `cloud.google.com/go/notebooks` using [DocFX](https://dotnet.github.io/docfx/).

## Prerequisites

- `godocfx` (generates the `obj/` folder)
- `docfx` (builds the documentation site)
- Python 3 (for local preview)

## Generate source docs

The `obj/` folder is generated by running:

```bash
godocfx cloud.google.com/go/notebooks
```

## Build the site

1. Create the `obj-gen/` directory and initialize DocFX inside it:

   ```bash
   mkdir -p obj-gen && cd obj-gen
   docfx init
   ```

2. Edit the generated `docfx.json` to add the path `obj/`.

3. Build the site:

   ```bash
   docfx build
   ```

   This creates the `obj-gen/_site/` folder.

## Preview locally

```bash
cd obj-gen/_site && python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080) in your browser.
